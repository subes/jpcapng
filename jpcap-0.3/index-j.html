Jpcap (Java package for libpcap)

(c) 2000 Keita Fujii

　

Jpcap はJavaからLAN上を流れるパケットの受信と任意のIPパケットの送信を可能とするJavaクラスパッケージです。
このパッケージはlibpcapとRaw Socket APIを使用します。

現在、FreeBSD 3.x, Linux RedHat6.1, Solaris, Windows 2000上での動作を確認しています。

対応プロトコルは、イーサネット, IPv4, IPv6, ARP/RARP, TCP, UDP, ICMPv4です。

インストール方法

＜Windows9x／NT／2000＞

  1. JDK 1.2以上をインストールします。
  2. libpcapのWindows移植版であるWinPcapのホームページからNDIS packet capture driverをダウンロードして
    インストールします。
  3. jpcap.zipをダウンロードして、適当なフォルダに展開します。
  4. Jpcap.dllをシステムフォルダ（\Windows\か\Winnt\system32）、またはパスの通っているフォルダに
    コピーします。
  5. jpcap.jarを\jdk1.?\jre\lib\extにコピーします。

＜UNIX＞

  1. 各プラットホームで利用可能な最新のJDKをインストールします。
  2. libpcapをインストールしていない場合はこちらからダウンロードしてインストールします。
  3. jpcap.tar.gzをダウンロードして展開します。
  4. jpcapディレクトリに移動し、Makefileを編集します。
  5. makeを実行します。
    もし "structure has no member named `sa_len'" というエラーがでてコンパイル出来ない時は、
    Jpcap_sub.hの "#define HAVE_SA_LEN"の1行をコメントアウトしてください。
  6. 出来上がったlibjpcap.soを
    JDK1.2の場合
       Javaのインストール先/jre/lib/extにコピーします。

    JDK1.1の場合
       Javaのインストール先/lib/プロセッサ名(i386)/スレッドライブラリ名(green_thread)にコピーするか、
       環境変数LD_LIBRARY_PATHで設定したパスに置いてください。

# Linux, FreeBSD, Solarisでコンパイルしたlibjpcap.soを同梱しておきます。それらを適切なディレクトリに
# コピーするだけで動作するかもしれませんが、保証はしません。
