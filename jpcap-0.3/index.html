Å@

Jpcap is a Java class package which enables to capture and send IP packets from Java application.
This package uses libpcap and Raw Socket API.

Currently, Jpcap is tested on FreeBSD 3.x, Linux RedHat 6.1, Solaris, and Microsoft Windows 2000.

Ethernet, IPv4, IPv6, ARP/RARP, TCP, UDP, ICMPv4 are supported.

How To Install

  <Windows 9x, NT, 2000>

  1. Install JDK1.2 or higher.
  2. Download and install NDIS packet captuer driver.
  3. Download and extract jpcap.zip
  4. Copy Jpcap.dll into system folder (\Windows or \Winnt\System32), OR
    Copy it to the folder where PATH is set.
  5. Copy jpcap.jar to Java installed dir\jre\lib\ext

  <UNIX>

  1. Install the latest JDK available on your platform.
  2. Download and install libpcap if not installed.
  3. Download and extract jpcap.tar.gz
  4. Move to jpcap directory, and edit Makefile
  5. Run make.
    If you get an error "structure has no member named `sa_len'",
    comment out the line "#define HAVE_SA_LEN" in Jpcap_sub.h.
  6. Copy libjpcap.so to

    JDK1.2 or higher
       Java installed dir/jre/lib/ext

    JDK1.1
       Java installed dir/lib/processor type (i386)/thread library name (green_thread)

# I include compiled libjpcap.so for Linux, FreeBSD, and Solaris. You may just copy it to the proper directory
# and use Jpcap. But I have no guarantee that they would work on your system.
