FAQ

Q: When I tried to run an application based on Jpcap, I got java.lang.UnsatisfiedLinkError. What should I do?

A: This error is caused when Java VM cannot find Jpcap library (Jpcap.dll or libjpcap.so). Please see Readme on where you should store the Jpcap library.

Q: Can I block or modify the packets instead of just caputuring them?

A: No. Jpcap only allows you to capture packets. In other words, the packets you captured by Jpcap are also transmitted to the destination hosts, and Jpcap cannot interfere the transmission.

Q: Can I capture packets from a WiFi (802.11) network interface?

A: The answer depends on what Operating System you are running and which WiFi devices your system has. On Windows, you may not be able to capture packets from some WiFi devices. Linux or BSD may have higher probability of supporting packet capturing using WiFi devices.
